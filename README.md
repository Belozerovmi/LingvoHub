📖 О проекте
LingvoHub — это frontend-проект лендинга онлайн-школы, который я разработал, чтобы продемонстрировать не только концепцию, но и реальную структуру современного веб-приложения. Проект целиком написан на нативных HTML5 и CSS3, без использования JavaScript-фреймворков, и представляет собой набор из 6 логически связанных страниц, каждая из которых решает свою пользовательскую задачу.

🗺️ Карта сайта и функционал
Страница	Файл	Назначение
Главная	index.html	Презентация платформы: УТП, популярные курсы, преимущества, процесс работы и блок регистрации.
Курсы	courses.html	Каталог курсов с фильтрацией по языку и карточками. Включает фильтры по уровню и целевой аудитории.
О нас	about-us.html	Развернутая информация о школе: миссия, аудитория, отличие от классических курсов.
Блог	blog.html	Раздел с новостями и статьями. Реализована сетка карточек с категориями и фильтром по типу контента.
Вход	enter.html	Форма авторизации с полями для логина и пароля.
Регистрация	registration.html	Форма регистрации по номеру телефона, а также альтернативные способы входа через Госуслуги, VK ID и Яндекс ID.
🎨 Дизайн и UI/UX
Единая дизайн-система: Все страницы объединены общей цветовой палитрой (акцентный фиолетовый #78258D, нейтральные фоны) и типографикой.

Адаптивная верстка: Интерфейс центрирован и имеет максимальную ширину в 1200px, что обеспечивает корректное отображение на различных экранах.

Микроанимации: Интерактивные элементы (кнопки, ссылки, карточки) имеют плавные переходы (transition) для отзывчивого интерфейса.

Кастомные элементы форм: Реализованы стилизованные чекбоксы, поля ввода с фокусом и кнопки в едином стиле.

🛠️ Технологический стек
Проект демонстрирует уверенное владение современными стандартами верстки и внимание к деталям.

Категория	Технология	Применение в проекте
Основы	HTML5, CSS3	Семантическая верстка, Flexbox, Grid Layout, кастомные формы, анимации.
Архитектура CSS	Разделение стилей	Для каждой страницы создан отдельный CSS-файл (courses.css, blog.css и т.д.), а общие стили вынесены в style.css.
Шрифты	Google Fonts	Подключены современные шрифты Inter, Manrope и Playfair Display для улучшения типографики.
Адаптивность	Резиновая верстка	Все блоки центрированы через display: flex и имеют фиксированную ширину контента, что упрощает дальнейшую адаптацию.
Деплой	GitHub Pages	Проект развернут и доступен по ссылке для демонстрации.
