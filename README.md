## 📖 О проекте

**LingvoHub** — это frontend-проект лендинга онлайн-школы, который я разработал, чтобы продемонстрировать не только концепцию, но и реальную структуру современного веб-приложения. Проект целиком написан на нативных **HTML5** и **CSS3**, без использования JavaScript-фреймворков, и представляет собой набор из **6 логически связанных страниц**, каждая из которых решает свою пользовательскую задачу.

### 🏠 Главная страница
> Файл: `index.html` | Стили: `style.css`

Презентация платформы, которая знакомит пользователя со всеми ключевыми преимуществами: УТП, популярные курсы, 3 причины быстрого освоения языка и пошаговый процесс начала обучения. Завершается блоком регистрации с формой для ввода телефона.

### 📚 Курсы
> Файл: `courses.html` | Стили: `courses.css`

Каталог языковых курсов с системой фильтрации. Пользователь может отфильтровать курсы по языку (немецкий, китайский, испанский), а также использовать сайдбар с чекбоксами для выбора уровня знаний и целевой аудитории. Карточки курсов отображают длительность, количество модулей и стоимость.

### ℹ️ О нас
> Файл: `about-us.html` | Стили: `about-us.css`

Развернутая информация о школе LingvoHub. Раздел включает миссию компании, описание целевой аудитории (от студентов до эмигрантов) и ключевые отличия от классических курсов — интерактивность, нескучный формат и свобода в выборе графика обучения.

### 📰 Блог
> Файл: `blog.html` | Стили: `blog.css`

Новостной раздел с фильтром по типу контента: все публикации, новости, статьи и интересное. Реализована сетка карточек статей с тегами, датами и превью текста. Выделенная "главная" новость отображается вверху, а остальные — в виде адаптивной сетки.

### 🔐 Вход
> Файл: `enter.html` | Стили: `enter.css`

Страница авторизации с минималистичной формой. Содержит поля для ввода логина и пароля, кнопку "Продолжить" и ссылку на страницу регистрации для новых пользователей.

### 📝 Регистрация
> Файл: `registration.html` | Стили: `registration.css`, `enter.css`

Форма регистрации с полем для ввода номера телефона (с маской). Особенность страницы — альтернативные способы входа через российские сервисы: **Госуслуги**, **ВКонтакте** и **Яндекс ID**. Каждый вариант стилизован в виде отдельной кнопки с логотипом сервиса.

## 🎨 Дизайн и UI/UX

- **Единая дизайн-система** — все страницы объединены акцентным фиолетовым цветом `#78258D`, нейтральными серыми фонами `#EFEFFF` и продуманной типографикой
- **Адаптивная верстка** — контент центрирован, максимальная ширина рабочей области — `1200px`
- **Микроанимации** — кнопки, ссылки и карточки имеют плавные переходы `transition` для отзывчивого UI
- **Кастомные формы** — стилизованные чекбоксы с анимацией, поля ввода с эффектом свечения при фокусе
- **Продуманная иерархия** — заголовки от 46px до 16px, аккуратная работа с прозрачностью `opacity: 0.6` для второстепенного текста
